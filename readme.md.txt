# Photolithography Through a Data Scientist's Eyes

Simulating photolithography imaging using Fourier optics in Python — mask patterns, diffraction, NA filtering, and image reconstruction.

## Contents

| File | Description |
|------|-------------|
| `photolithography_fourier_optics.ipynb` | Complete Python notebook — run in Google Colab |
| `photolithography_section1.docx` | Article Section 1 — What is Photolithography |
| `photolithography_section2.docx` | Article Section 2 — Physics of Light |
| `photolithography_section3.docx` | Article Section 3 — Wavelength and NA Limits + Superconductivity |
| `photolithography_section4.docx` | Article Section 4 — Fourier Optics |

## Run the Notebook

Open `photolithography_fourier_optics.ipynb` in Google Colab — no installs needed, pure numpy and matplotlib.
